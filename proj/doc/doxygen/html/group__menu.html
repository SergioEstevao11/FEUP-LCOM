◆ incrementGameTick()

void incrementGameTick ( Game *  game)

Function that increments the chronometer in every interrupt of the timer.

Since 1 second divided by 30hz(game's refresh rate) is a irracional number (0.0333(3)), the tick increment is done in a 3-3-4 order, because incrementing a infinite decimal number in every timer interrupt could cause to rounding erros by the program

Parameters
gameVariable where the game time is saved
